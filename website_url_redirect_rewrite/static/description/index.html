SEO URL Redirect / Rewrite for 301, Blog, E-commerce website or any URL

Features

We have added below mentioned features for this URL Redirect or URL Rewrite module

SEO Friendly URL

Make URL SEO friendly set redirection or rewrite URL

Source to Destination

Redirect from source URL to redirected URL for your website

Long URL to Short

Redirect blog long URL to rewrite short URL

Remove / Edit / Delete

Remove, Edit and Delete Source and Destination URL

Suffix URL Redirect

Add suffix URL for any URL.

Redirect Automatic

Automatically redirection apply once set URL from destination

Set Bulk Redirect

Add bulk rewrite or redirect URL

Apply for Categories

Redirect or Rewrite URL for Product categories/ Sub categories

This module allows to set redirection or rewrite URL in odoo using Following Five Steps:

Single and Bulk URL Redirect

Click on SEO URL Redirect for single page URL redirect and
Click on product and category wise URL redirect/rewrite for Bulk or multiple URL


Manage SEO URL redirect

Category wise and Product wise, List of URLs are appear here for edit, delete and add


Set Source and Destination URL

Set Original URL, Destination URL and select URL type


Product URL Rewrite for Single/Bulk URL Rewrite

Tick mark all those url which you want to redirect and click on Update SEO URL Products "


URL Redirect Sucessfully

After clicked on Update SEO URL Products button - message will display with number of URL redirected

This URL redirection/rewrite module implemented for og:url property (Meta property : Open graph URL)

External python library 'pyotp' is required.

To install it in odoo 09,10 use command: sudo pip install pyotp

To install it in odoo 11,12,13 use command: sudo pip3 install pyotp
